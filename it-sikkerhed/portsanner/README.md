# Port Scannner


## Required modules
You have to install

  pip3 install colorama
 
 
 ## How to run
 Use this:
 
  python3 fast_port_scanner.py <IP> --ports 1-1000
 
 Like this
 
  python3 fast_port_scanner.py 94.18.243.162 --ports 1-1000
  
  or
 
  python3 fast_port_scanner.py 192.168.1.1 --ports 1-1000
